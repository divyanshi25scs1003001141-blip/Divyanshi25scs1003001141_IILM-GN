Dataset Splitting & Model Evaluation

This project demonstrates different techniques for splitting a dataset into training and testing parts and evaluates model performance.

🚀 Features
Train-Test Split (70-30)
Cross-Validation (10-fold)
Leave-One-Out Cross-Validation (LOO)
Random Shuffle Split
Bootstrap Sampling
🤖 Models Used
Logistic Regression
Decision Tree
K-Nearest Neighbors (KNN)
⚙️ Workflow
Load dataset
Define models
Apply different splitting techniques
Measure accuracy and execution time
🎯 Goal
To compare different data splitting methods and analyze their impact on model performance.
